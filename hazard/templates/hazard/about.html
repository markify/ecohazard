About

This web app was built by software engineering students in SFSU.
It provided knowledge of software engineer practices and development cycle in a real work environment.
The goal was to solve issues in the environment by developing an environmental reporting web application.
It was developed and presented during Spring of 2018.
Currently, it is being developed and redesigned by mark@ecohazard.me
If you would like to contribute new features, organize, or bug hunt. It is open source Github
